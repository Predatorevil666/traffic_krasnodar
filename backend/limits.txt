# Информация о лимитах запросов API 2GIS MapGL

По результатам анализа документации 2GIS и поиска в интернете, могу предоставить следующую информацию о лимитах запросов:

## Лимиты по количеству запросов:

Для бесплатного использования API MapGL существуют следующие ограничения:

1. Лимит запросов в секунду: 50 запросов в секунду

2. Лимит запросов в сутки: отсутствует явное ограничение для базового API MapGL 
   (Примечание: для некоторых связанных API, например Directions API, могут быть отдельные лимиты)

Как и для многих других картографических API, превышение лимита скорости запросов (50 в секунду) 
может привести к временному блокированию доступа или получению ошибок при запросах.

## Дополнительные ограничения:

- Для показа карт в коммерческих приложениях может потребоваться приобретение платной подписки
- При интенсивном использовании API рекомендуется обратиться в службу поддержки 2GIS для уточнения лимитов и условий использования

## Источники:
Эта информация получена на основе доступной документации 2GIS и общей информации о работе API.
Точные ограничения могут изменяться, поэтому для получения актуальной информации рекомендуется 
обратиться напрямую в службу поддержки 2GIS.
